# Evaluating the Trustworthiness and Security of AI-Generated JavaScript Code

This repository contains the code and configuration used in the master’s thesis:

**“Evaluating the Trustworthiness and Security of AI-Generated JavaScript Code”**  
Author: Amirhossein Sedigh Daemi (MSc Computer Science)

The project analyzes AI-generated JavaScript and TypeScript code from public GitHub repositories using static analysis (CodeQL) to identify security weaknesses.

The entire pipeline, including data collection, preprocessing, attribution, and analysis, is fully automated.
